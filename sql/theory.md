# Introduction to Databases

Modern applications handle large amounts of data.

Examples of data include:

- User accounts
- Messages
- Photos and videos
- Orders and transactions
- Product information

Applications need a structured way to **store, organize, and retrieve this data**.

This is where databases are used.

---

## What is a Database?

A **database** is an organized collection of data that can be stored, accessed, and managed efficiently.

Databases allow applications to store large amounts of information and retrieve it quickly when needed.

Examples of data stored in databases include:

- User profiles
- Login credentials
- Product catalogs
- Transaction records
- Messages and posts

Almost every modern software application relies on databases.

---

## Why Databases are Important

Without databases, managing large amounts of data becomes difficult.

Databases help applications:

- Store data in an organized way
- Retrieve data quickly
- Update and manage information efficiently
- Maintain consistency of data

Databases are essential for applications that handle thousands or millions of users.

---

## Examples of Databases in Real Applications

Different applications store different types of information in databases.

Examples:

Instagram stores:

- User accounts
- Photos and videos
- Comments
- Followers and following lists

Amazon stores:

- Product information
- Customer accounts
- Orders and payments
- Inventory data

Banking systems store:

- Customer details
- Account balances
- Transaction records

---

## Database Management System (DBMS)

A **Database Management System (DBMS)** is software used to create, manage, and interact with databases.

DBMS provides tools to:

- Store data
- Retrieve data
- Update data
- Delete data

Examples of DBMS include:

- MySQL
- PostgreSQL
- Oracle
- Microsoft SQL Server

These systems allow developers to manage data efficiently.

---

## Tables in a Database

Data in relational databases is stored in **tables**.

A table organizes data into rows and columns.

Example table: Users

| ID | Name | Email |
|----|------|------|
| 1 | Rahul | rahul@email.com |
| 2 | Priya | priya@email.com |
| 3 | Aman | aman@email.com |

Each row represents a **record**.

Each column represents a **field** that describes the data.

---

## Types of Databases

Databases can be categorized into different types depending on how data is stored.

### Relational Databases

Relational databases store data in structured tables with rows and columns.

Examples:

- MySQL
- PostgreSQL
- Oracle

Relational databases use **SQL (Structured Query Language)** to interact with data.

---

### NoSQL Databases

NoSQL databases store data in more flexible formats.

Examples include:

- MongoDB
- Cassandra
- Redis

NoSQL databases are often used for large-scale applications and flexible data structures.

---

## Where Databases Are Used

Databases are used in almost every digital system.

Examples include:

- Social media platforms
- Online shopping websites

Whenever an application needs to store and retrieve information, a database is usually involved.
